Exemplos de formatações de texto

Negrito / Destaque

Para tornar uma parte de frase em negrito usa-se a tag <b>

Agora para tornar uma parte da frase em destaque usa-se a tag <strong>
Não se esqueça da Sematinca

Italico / Enfase

Nessa frase tera uma parte em italico que se utiliza a tag <i>

Nessa frase existe uma parte que esta com enfase utilizando a tag <em>

Texto marcado

Pode-se marca uma parte do texto usando a tag <mark>

Texto grande e pequeno

Texto grande e texto pequeno tags <big> e <small>

Texto deltado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado

texto inserido

Podemos marca um texto como inserido para dar uma enfasse e indicar que ele foi adicionado depois
